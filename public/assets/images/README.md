Place your real images in these folders and keep the same filenames.

Required files used by website:

- `logo/logo.svg` (brand logo)
- `home/hero.svg` (home page)
- `products/cover.svg` (products page)
- `design/cover.svg` (design page)
- `about/cover.svg` (about page)
- `contact/cover.svg` (contact page)

You can replace each `.svg` file with `.jpg` or `.png` if needed.  
If you change file extension or name, also update paths in `src/App.tsx`.
